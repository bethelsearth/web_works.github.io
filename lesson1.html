1. Introduction to Web Technology

1. Introduction to Web Technology

Overview: Understanding how devices communicate using markup languages and the role of the WWW.

Core Concepts Covered:

  • WWW: A collection of linked pages across the globe.
  • Web Pages vs. Websites: A website is a collection of interlinked web pages.
  • URLs: Unique addresses (Uniform Resource Locators) for locating web resources.
  • Web Servers: High-end computers or cloud services that host websites.
2. HTML Structure & Elements

2. HTML Structure & Elements

Overview: Mastering the structural foundation of the web using tags and attributes.

What I Learned:

  • Semantic Tags: Using &lt;header&gt;, &lt;main&gt;, &lt;footer&gt;, and &lt;article&gt; for accessibility.
  • Text Formatting: Headings (H1-H6), lists (ordered vs. unordered), and anchors for navigation.
  • Forms & Tables: Creating data entry points (inputs, radio buttons, dropdowns) and structured data displays.
  • Multimedia: Inserting images with alt text for screen readers.
3. PHP Control Structures

3. PHP Control Structures

Overview: Learning how scripts respond differently to various inputs and situations.

Key Logic:

  • Conditional Statements: Branching execution paths based on truth values.
  • Types: if, if-else, if-elseif-else, and the switch statement for multiple constant values.
  • Comparison: Using operators to check conditions (e.g., $age < 30).
4. Browser Cookies in PHP

4. Browser Cookies in PHP

Overview: Identifying users and storing small pieces of data (up to 4KB) on the client side.

Functional Knowledge:

  • setcookie(): Function used to embed small files on the user's computer.
  • Parameters: Name, Value, Expiry, Path, Domain, and Security.
  • Usage: Personalizing pages and tracking user information across sessions.
  • Constraints: Cookies must be sent before any HTML output.
